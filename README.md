## 🖥️ Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/AjayNaik01/vivid-glide-site.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## 📦 Build

To build for production:

```sh
npm run build
```
